   - How to do this efficiently for 100+ chunks?
   - Should I use spatial indexing (quadtree)?

4. **Color Generation**:

   - Golden ratio method?
   - Show algorithm with code

5. **Performance Optimization**:

   - Should I use offscreen canvas?
